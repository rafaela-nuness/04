Projeto 3: Sistemas operacionais – 12/02

Screenshot/mockup do projeto

Descrição técnica

O projeto envolveu a exploração prática de um sistema Linux rodando no VirtualBox, realizando tarefas como navegação no sistema de arquivos, uso do terminal para comandos básicos, instalação de programas e navegação na internet. Em seguida, foi feita uma comparação entre Linux e Windows, analisando interface, sistema de arquivos, terminal, gerenciamento de software, segurança, desempenho e curva de aprendizado.

Desafios enfrentados

Adaptar-se a um novo sistema operacional, aprender comandos básicos do terminal e compreender a diferença na estrutura dos sistemas de arquivos e permissões.

Aprendizados adquiridos

Melhor compreensão do funcionamento do Linux, reconhecimento das diferenças e semelhanças com o Windows, e percepção das vantagens do Linux em segurança e desempenho para determinados usos.
